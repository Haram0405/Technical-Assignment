# Manual HTML Project

This repository contains two HTML projects created manually without copying AI-generated code.
The purpose of this task was to practice writing HTML structure by hand and improve understanding of basic web development concepts.

---

# Folder Structure

```bash
manual-html/
│
├── survey-form.html
├── ecommerce-product.html

```

---

# Project 01 – Survey Form

## Overview

The `survey-form.html` file is a simple survey form inspired by freeCodeCamp's survey page.
It collects user information using different HTML form elements.

---

# Features Used

* Heading and paragraph tags
* Form element
* Input fields
* Labels
* Radio buttons
* Checkboxes
* Dropdown menus
* Required fields
* Basic alignment using HTML attributes

---

# Code Structure

## Form Section

The entire form is wrapped inside the `<form>` tag.

### Purpose:

* Groups all form elements together
* Makes the form organized and structured

---

## Input Fields

Used for:

* Name
* Email
* Age

### Purpose:

* Collect user information
* Practice different input types like:

  * `text`
  * `email`
  * `number`

---

## Dropdown Menus

Dropdowns were used to allow users to select:

* Current role
* Favorite feature

### Purpose:

* Makes form interaction easier
* Prevents unnecessary typing

---

## Radio Buttons

Used for recommendation selection.

### Purpose:

* Allows users to select only one option

---

## Checkboxes

Used for improvement suggestions.

### Purpose:

* Allows selecting multiple options

---

# Design Decisions

* Used centered alignment for a simple layout
* Added spacing using `<br>` tags for readability
* Kept the design clean and beginner-friendly
* Focused more on HTML structure rather than advanced styling
