# Luxury Watch Product Page

This project is a modern ecommerce product page created using HTML, CSS, and JavaScript.
The design is inspired by luxury watch ecommerce websites with a clean layout, dark navbar, product gallery, pricing section, and interactive quantity buttons.

---

# Prompts Used

## Prompt 01 – Ecommerce Product Page Design

### WHAT

Build a luxury watch ecommerce product page.

### HOW

Use a dark aesthetic theme with a clean layout, attractive visuals, responsive structure, and premium UI design.

### WHY

I am a frontend developer and wanted to practice building a modern ecommerce product page with a professional user experience.
